# **App Name**: b.max.dev Portfolio

## Core Features:

- Dynamic Alias Display: A prominent and engaging display for the 'b.max.dev' alias, potentially featuring subtle animations or interactive elements.
- Section Navigation: Intuitive navigation to easily access and browse different portfolio categories: 'Continuous Works', 'Build Projects', 'Skills Learning', 'Works', and 'Hobbies'.
- Interactive Item Cards: Display projects and items within each category using responsive, interactive cards with hover effects for engaging previews.
- AI-Generated Summaries: A tool that generates concise and impactful summaries for projects or work entries, enhancing the descriptive text on item cards.
- Detailed Project Views: Clicking an item card reveals a dedicated detail view or page for the project, offering more comprehensive information, images, and links.
- Responsive Layout: Ensures the portfolio is visually appealing and fully functional across all device types and screen sizes.

## Style Guidelines:

- A dark color scheme evokes sophistication and modernity. The primary color, a desaturated dark blue-gray (HSL(220, 20%, 30%), #3D474D), serves as a grounding base for content areas. The background, a very dark charcoal with a cool undertone (HSL(220, 10%, 10%), #17191A), provides the requested dark mode ambiance. A vibrant electric blue (HSL(210, 100%, 50%), #0080FF) acts as the accent color for key interactive elements and highlights, bringing energy to the sleek design.
- Headline font: 'Space Grotesk' (sans-serif) for its modern, techy, and precise feel, ideal for prominent titles and the alias display. Body font: 'Inter' (sans-serif) provides excellent readability and a contemporary, objective aesthetic for all textual content within project descriptions and details.
- Use a set of minimalist, vector-based icons that align with the clean, modern aesthetic, enhancing navigation and feature recognition without visual clutter.
- A structured and clean layout with ample whitespace to ensure focus on the portfolio items. Utilize a grid-based system for organizing project cards, promoting visual balance and easy scannability across categories.
- Implement subtle, fluid animations for interactive elements such as card hovers, section transitions, and menu expansions. These should be fast and non-disruptive, adding to the 'sleek' and 'dynamic' user experience.
